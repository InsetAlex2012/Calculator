## ‚ú® Features

### Calculator Functions
- Basic arithmetic operations: `+`, `-`, `*`, `/`
- Parentheses for complex expressions: `()`, `[ ]`, `{ }`
- Exponentiation with `Power` button or `p`/`P` keys
- Real-time input via buttons or keyboard

- Built with **Tkinter**
- Styled buttons for numbers, operations, parentheses, and power
- Large display using `Entry` widget
- Responsive layout with `grid` configuration
- Custom window icon (`math.png`)
- Background color: `#DADBDD`
